New side project: Familie Heist

16 September 2021 | 1 minute read

Something for everyone

The word Heist comes from Heist-op-den-Berg, a town in Belgium. The idea behind this project is to provide something like a wiki, but more user-friendly. We're going to build something in order to remember our family in a more visually entertaining and informative way.

I wanted to put as little effort in it as possible since we're short on time. So far we've tried both MediaWiki and Drupal. The user experience from MediaWiki was uninviting for several people from our family. And Drupal offered an unstable development experience, being largely database driven.

So, back to square one. Let's build it from scratch. This time using Spring and Angular. If you're interested, the source code can be found at Github.
